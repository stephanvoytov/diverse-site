# Diverse — Лендинг франшизы в России

Официальный лендинг представителя бренда **Diverse** в России (ООО «Хаус»).

Сайт: [diversebrand.ru](https://diversebrand.ru)

---

## О проекте

Лендинг франшизы магазинов одежды Diverse. Бренд основан в 1993 году в Гданьске (Польша), насчитывает 400+ магазинов в 9 странах. ООО «Хаус» (Калининград) — эксклюзивный дистрибьютор в РФ и СНГ, уже открыл 11 магазинов.

### Страницы

| Маршрут | Контент |
|---------|---------|
| `/` | Главная: герой, бренд в цифрах, матрица, франшиза, карта, контакты |
| `/about` | История Diverse, философия, 3 линии (Diverse / DEXT / Coalition), таймлайн |
| `/collection` | Коллекции бренда: full-screen параллакс-секции с описаниями |
| `/franchise` | Условия франшизы: 3 варианта, сравнение, преимущества, форма |
| `/stores` | Магазины: интерактивная карта и фото 11 точек в РФ и Казахстане |

## Стек

| Технология | Назначение |
|------------|------------|
| Next.js 16 (App Router) | Фреймворк |
| TypeScript | Типизация |
| Tailwind CSS v4 | Стилизация |
| Framer Motion | Анимации, параллакс |
| Leaflet + MarkerCluster | Интерактивная карта магазинов |
| React Hook Form + Zod | Форма обратной связи |

## Запуск локально

```bash
npm install
npm run dev
# → http://localhost:3000
```

Сборка:

```bash
npm run build
npm run start
```

```
src/
├── app/
│   ├── about/           # /about
│   ├── collection/      # /collection
│   ├── franchise/       # /franchise
│   ├── stores/          # /stores
│   ├── layout.tsx       # корневой layout (шрифты, метаданные)
│   ├── page.tsx         # главная
│   └── globals.css      # Tailwind theme
├── components/
│   ├── blocks/          # секции главной (Hero, About, Franchise, …)
│   └── shared/          # Header, Footer, StoresMap
└── data/                # контент (коллекции, магазины, навигация, …)
```

## Деплой

Сайт статический (Next.js `output: 'export'`). Деплоится на GitHub Pages через GitHub Actions при пуше в `main`.

Вручную:

```bash
npm run build
# → out/ — готовые файлы
```

## Важно

- Все материалы — официальная информация ООО «Хаус».
- Бренд Diverse принадлежит ETOS S.A. (Польша). ООО «Хаус» — официальный дистрибьютор.
- Координаты магазинов в `src/data/stores.ts` хранятся в формате `[lng, lat]`.
