**— Отсутствие дополнительных платежей по франшизе**
Роялти — НЕТ. Паушального взноса — НЕТ. Предоплат — НЕТ.
Оплачивается только товар, подготовленный к отправке.

**— Гарантия уникальности**
Только Вы будете развивать сеть магазинов "Diverse" в вашем городе.

**— Высокая маржинальность**
Мы даем возможность зарабатывать нашим партнерам, чтобы спокойно покрывать затраты и получать хорошие дивиденты.

**— Заказы товара с головного склада**
Заказы производятся со склада в Гданьске (Польша) через B2B-платформу с личным кабинетом. Доступ к огромному складу + предзаказы коллекций 6 раз в год.

**— Регулярные поставки товара**
Поставки раз в месяц: сбор заказов → упаковка в Гданьске → таможня в Калининграде → доставка EMS в ваш город.

**— Сопровождение проекта 24/7**
От подбора локации, дизайн-проекта, выбора коллекции, обучения персонала до текущей поддержки.

**— Выход на рынок с европейской маркой**
Актуальные модели, огромный ассортимент и высокое качество.

**— Полная легальность**
Сертификация ЕАС + система "Честный Знак".

### 4.5. Логистика

## 8. ДИЗАЙН-СИСТЕМА

На основе анализа официальных сайтов:
- **diversesystem.com** — основной бренд (urban minimal, quiet luxury)
- **diverseextremeteam.com** — экстрим-линия (motorsport, агрессия)

Для лендинга франшизы берём стилистику **основного бренда** с возможными акцентами DEXT в секции про Dakar.

### 8.1. Шрифт

```css
Inter (300, 400, 500, 600, 700)  /* как на официальном сайте */

Заголовки:   700, ВСЕ ПРОПИСНЫЕ
Подзаголовки: 500, normal case
Текст:        400, normal case
Мелкий текст: 300, normal case
```

### 8.2. Цветовая палитра

```
Фон основной:    #FFFFFF (белый)
Фон секций:      #F5F5F5 (светло-серый, для чередования)
Фон тёмный:      #000000 (чёрный — хедер, футер)
Фон карточек:    #1A1A1A / #191919 (тёмный для фото с текстом)

Текст основной:  #1A1A1A (почти чёрный)
Текст серый:     #666666 (второстепенный)
Текст на тёмном: #FFFFFF (белый)
Текст ссылки:    #666666 (серый, 14px, weight 300)

Акцент:          #D12026 / #DC2626 (красный — кнопки, выделения)
Акцент DEXT:     #FF0000 (ярко-красный — только для Dakar/спорт)
```

### 8.3. Типографика

| Элемент | Размер | Weight | Регистр |
|---------|--------|--------|---------|
| Hero-заголовок | 56-72px | 700 | ПРОПИСНЫЕ |
| Заголовок секции | 36-48px | 700 | ПРОПИСНЫЕ |
| Подзаголовок | 20-24px | 500 | normal |
| Основной текст | 16-18px | 400 | normal |
| Мелкий текст | 14px | 300 | normal |

### 8.4. Стиль и настроение

```
✓ Минимализм — много белого/серого пространства
✓ Моновпиксельные линии как разделители (без теней)

Кнопки:
```
